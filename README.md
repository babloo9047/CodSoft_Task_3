# ATM Interface (Java)

## 📌 Project Overview
The **ATM Interface** is a Java-based console application that simulates the basic functionalities of an Automated Teller Machine (ATM). It allows users to securely access their account using a PIN and perform essential banking operations such as checking balance, depositing money, withdrawing cash, viewing transaction history, and changing the PIN.

This project was developed as part of the **Oasis Infobyte Java Development Internship**.

---

## 🚀 Features
- Secure PIN Authentication
- Balance Inquiry
- Cash Deposit
- Cash Withdrawal
- Transaction History
- PIN Change Option
- Menu-Driven Interface
- Input Validation
- User-Friendly Console Interface

## ▶ How to Run
1. Clone or download this repository.
2. Open the project in your preferred Java IDE (IntelliJ IDEA, Eclipse, or VS Code).
3. Compile the Java file.
4. Run the program.
5. Enter the correct ATM PIN to access the system.
6. Select the desired banking operation from the menu.
